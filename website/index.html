What You Get

Ed25519 Identity

Unique Pillar ID derived from your keypair. Every response is cryptographically signed.

Mesh Networking

Auto-discover peers via UDP multicast. Replicate registries. Join the global mesh.

Wallet Auth (SIWE)

Sign-In with Ethereum. Browser extension with EIP-6963 multi-wallet support.

EVM Gateway

Built-in JSON-RPC proxy for 5 chains with automatic failover.

Encrypted Vault

AES-256-GCM encrypted storage. Shamir secret sharing for key recovery.

Tor Support

Optional .onion hidden service. Access your Pillar anonymously.
