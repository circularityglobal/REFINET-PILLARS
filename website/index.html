One node. Full stack.

#>
ED25519 IDENTITY
Keypair generated on first run. PID = SHA-256(public key). Every response signed and verifiable.
~~
MESH NETWORKING
UDP multicast peer discovery on LAN. WAN via peers.json bootstrap. Signed registry replication. No central server.
Оһ
WALLET AUTH
Sign-In with Ethereum (EIP-4361). Multi-wallet via EIP-6963. Chrome extension v0.4.0 with one-click signing.
в¬Ў
EVM GATEWAY
JSON-RPC proxy for 5 chains. Ethereum, Polygon, Arbitrum, Base, Sepolia. Automatic failover. SIWE-gated broadcast.
в–Ј
ENCRYPTED VAULT
AES-256-GCM storage. Argon2id key derivation. Shamir secret sharing. SHA-256 named files, no metadata leakage.
@
TOR HIDDEN SERVICE
Optional .onion address via stem + Tor binary. Anonymous inbound access. No port forwarding needed.
//
GOPHERHOLE HOSTING
Immutable, append-only registry with Ed25519 signatures. Registrations replicate across the entire mesh automatically.
13
13-MONTH LEDGER
SQLite with 13Г—28-day rolling window + yearly archive. WAL mode. Deterministic accounting periods. No cloud.
